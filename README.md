&#x20;

\## Gameplay

&#x20;

The player controls a pixel-art samurai cat navigating upward through cloud platforms while avoiding enemies and collecting coins. The goal of each level is to reach the finish point at the top.

&#x20;

Controls:

&#x20;

| Key | Action |

|-----|--------|

| A / D | Move left / right |

| Space | Jump |

| J | Boost |

| Tab | Toggle control hint |

| Escape | Pause |
