## Gameplay

Control a pixel-art samurai cat as it climbs through the skies, jumping across cloud platforms, avoiding enemies, and collecting coins.

The objective of each level is to reach the finish point at the top.

### Controls

| Key | Action |
|------|--------|
| A / D | Move left / right |
| Space | Jump |
| J | Boost |
| Tab | Toggle control hints |
| Escape | Pause |
